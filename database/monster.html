# 结构

字段 数据类型 默认值 空 属性 索引 A_I 注释
id BIGINT UNSIGNED PRIMARY 是 ID
title VARCHAR(32) INDEX 怪物名
image VARCHAR(256) '' 图像
description VARCHAR(256) '' 描述
dialogue VARCHAR(256) '' 对话
lv MEDIUMINT 1 UNSIGNED 等级
race TINYINT UNSIGNED 种族
charac TINYINT UNSIGNED 特性
career TINYINT UNSIGNED 职业
  1. race: 1: 不死族,2: 兽族, 3: 高级精灵, 4: 其他.
  2. charac: 1: 人形态, 2: 血牛, 3: 猛兽, 4: 其他.
  3. career: 1: 战士, 2: 法师, 3: 射手.
