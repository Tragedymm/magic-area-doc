# 分区细节

  1. 每个区采用独立的虚拟主机/数据库/域名[/端口],同一台服务器也可以开启若干分区.
  2. 分区配置需要在配置文件单独写入,不能通过数据库配置.
  3. 每个账号在同一个区最多可以创建3个角色.

# 合区冲突解决方案

  1. 主键冲突,主键统一用BIGINT UNSIGNED数据类型,最大2^64-1,预留一百万亿的自增变化空间,低风险高效率合区.
  2. 名称冲突,重名分别修改名字为名字1, 名字2, 名字3....
  3. 合区后角色数量大于3的,多的角色不会被删除,但是不能创建新的角色,除非手动删除角色至数量少于3.
