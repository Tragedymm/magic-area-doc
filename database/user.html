# 用户表

# 属性

表名 注释 排序规则 存储引擎
users 用户表 utf8mb4_bin InnoDB

# 结构

字段 数据类型 默认值 空 属性 索引 A_I 注释
id BIGINT UNSIGNED PRIMARY 是 ID
username VARCHAR(16) UNIQUE 用户名
password_hash CHAR(64) 密码哈希
token CHAR(128) UNIQUE 临时身份认证
email VARCHAR(128) UNIQUE 邮箱
tel CHAR(11) UNIQUE 手机号
reg_ip VARCHAR(39) '' 注册IP
reg_time INT 0 UNSIGNED 注册时间
login_ip VARCHAR(39) '' 最后登录IP
login_time INT 0 UNSIGNED 最后登录时间
is_ban TINYINT 0 UNSIGNED 是否封禁
is_freeze TINYINT 0 UNSIGNED 是否冻结
unfreeze_code CHAR(6) '' 解冻验证码
balance DECIMAL(10, 2) 0.00 UNSIGNED 余额
recharge DECIMAL(10, 2) 0.00 UNSIGNED 充值总额
  1. is_ban: 1: 是, 2: 否.
  2. is_freeze: 1: 是, 2: 否.
