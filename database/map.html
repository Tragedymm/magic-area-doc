# 结构

字段 数据类型 默认值 空 属性 索引 A_I 注释
id BIGINT UNSIGNED PRIMARY 是 ID
title VARCHAR(32) INDEX 地图名
image VARCHAR(256) '' 图像
description VARCHAR(256) '' 描述
type TINYINT UNSIGNED 类型
flush_interval INT 0 UNSIGNED 刷新间隔
north BIGINT 0 UNSIGNED 北出口
west BIGINT 0 UNSIGNED 西出口
east BIGINT 0 UNSIGNED 东出口
south BIGINT 0 UNSIGNED 南出口
monster JSON [] 怪物
npc JSON [] NPC
func JSON [] 功能
delivery JSON [] 传送
  1. type: 1: 城镇, 2: 野外, 3: 副本, 4: 其他.
  2. flush_interval: 0 为不设定，表级配置优先级最高.
  3. monster: JSON 数组. 若干对象组成, 每个对象包含怪物ID和怪物数量两个值.

    {

]
