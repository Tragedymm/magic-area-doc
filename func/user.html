# 登录

  1. 默认使用账号（用户名）登录，也可以手动选择使用邮箱/手机号登录。
  2. 需要输入验证码，防止恶意请求数据库。
  3. 开启 CSRF 验证，防止跨站攻击。
  4. 密码错误 5 次，冻结账号，需要邮箱/手机号验证解锁，防止暴力试错。

# 效果

# 注册

  1. 账号（用户名）为6-16位字母或者数字，不区分大小写。
  2. 密码为6-16位字母或者数字，不区分大小写。
  3. 密码不能与账号或手机号相同。
  4. 密码不能过于简单，如：111111、123456、abc123、abcdef等。
  5. 需要输入验证码，防止恶意请求数据库。
  6. 开启 CSRF 验证，防止跨站攻击。
  7. 每个 IP 每天只能注册 3 个账号，防止批量刷账号。
  8. 一个邮箱/手机号只能绑定一个账号。
  9. 邮件/短信发送间隔为30分钟，防止恶意刷邮件/短信。

# 效果

# 找回/重置密码

  1. 密码为6-16位字母或者数字，不区分大小写。
  2. 需要输入验证码，防止恶意请求数据库。
  3. 开启 CSRF 验证，防止跨站攻击。
  4. 邮件/短信发送间隔为30分钟，防止恶意刷邮件/短信。

# 效果

# 修改密码

  1. 新密码不能与旧密码相同。
  2. 新密码不能与账号或手机号相同。
  3. 密码为6-16位字母或者数字，不区分大小写。
  4. 修改密码后自动退出。

# 效果

# 充值

  1. 卡密充值,有10元/50元/100/500元等的卡密.
  2. 充值输入卡号和卡密,卡号和卡密都为32位.
  3. 每个卡密只能使用一次.

# 效果

# 查看充值历史

  1. 查看所有的历史充值记录.
  2. 充值记录包括卡号,充值时间,金额,状态.

# 效果

# 查看消费流水

  1. 显示所有的消费历史记录.
  2. 消费流水包括区服,角色,消费项目,金额,时间.

# 效果

# 退出登录

  1. 退出清理所有COOKIE和SESSION.
  2. 账号和所有角色相关缓存清空.
  3. 所有区服角色强制下线.

# 生成书签

  1. 生成书签可以生成免输账号的快速登录链接.
  2. 书签链接有效期为30天,30天后需要重新登录生成书签.
  3. 退出登录会使书签失效.
  4. 同时只能有一个书签有效.

# 让之前的书签失效

  1. 会使有效的书签立刻失效.

# 查看登录历史

  1. 可以查看账号的登陆历史.
  2. 包括书签登录和输入账号密码登录.
  3. 登陆历史可以查看登录IP,时间.

# 效果

# 修改邮箱/手机号

  1. 修改手机号或者邮箱需要验证当前的邮箱/手机号.
  2. 也需要验证新的的邮箱/手机号.
  3. 丢失当前的邮箱/手机号可以联系管理员修改.
  4. 修改成功后的效果等同于退出登录的效果.
